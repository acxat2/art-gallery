# ArtStudio || Art-Family
<p>Это приложение для художников и тех, кому интересны их работы (https://art-nest-gallery-space.web.app). Здесь присутствует полноценный backend, написаный на Node.js Express.js., позволяющий проходить регистрацию, аутентификацию, загружать свои рисунки и публиковать те, что считают нужным, для общего просмотра, а так же просматривать работы других художников</p>

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 17.1.3.
